eval 函数

  • 语法 eval(符合JS语法规范的字符串)
  • 作用：在代码执行过程中，将eval里面的代码执行

Function 和 eval 的异同点

  1. 都可以将字符串转换成代码

  2. Function转换成代码后仍是函数，需要手动执行。eval在代码执行过程中就执行了。

Function 和 eval 的问题

  1. 执行效率问题

  2. 安全性问题 (xss脚本攻击)

Function原型链

  • JS中的所有的构造函数都是Function，包括Function自己

  • Function是自己的构造函数，也是自己的实例

  • Function构造函数在原型中担任两个角色，一个是构造函数，一个是实例。

  • 在作为构造函数时，访问原型使用Function.prototype

  • 在作为实例时，访问原型使用Function.__proto__
