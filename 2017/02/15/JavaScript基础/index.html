变量什么时候是undefined

  • 变量声明后未赋值

  • 手动赋值为undefined

  • 当一个对象的属性并不存在的时候，会返回一个undefined

变量什么时候是null

  • 变量不可能自动成为null，需要人为的去赋值

  • 当为别人提供一个返回对象的方法或函数的时候，函数或者方法创建失败，或者获取失败，无法返回一个正常的方法或函数，就会返回一个null

  • 当我们想为下面的某一个对象提供一个全局对象名的时候，在刚开始可以定义一个变量，用null来占位。

  1. for in遍历 遍历的是对象的键

    • 在for-in中使用键来访问对象的值的时候，不能使用点语法，只能使用中括号，即关联数组语法。

值类型和引用类型

概念

  1. 值类型

    • 直接存储数据本身的数据，就是值类型的数据

  2. 引用类型

    • 存储的是数据的地址，数据单独开辟一块空间自行存储，这就是引用类型的数据

赋值特征

  1. 值类型

    • 值类型的数据在赋值的时候，会将存储的数据复制一份交给新的数据，两个变量值相等。当在修改其中任意一个数据的时候，另一个数据的值并不会发生改变。

  2. 引用类型

    • 引用类型的数据在赋值的时候，会将对象的地址复制一份交给新的变量，此刻两个变量指向同一个地址。通过其中任意一个变量对其对象进行修改，另一个也会受到影响

在JavaScript中 没有堆和栈的概念

对象的动态特性

指的是，在对象创建出来后，可以随时为对象添加成员的这个特性，就是动态特性。

delete 关键字

  • 语法：delete 变量名\/对象名.属性名

  • 返回值：boolean类型

    1. 删除未使用var声明的变量

    2. 删除对象的属性

  • delete删除数组元素时，数组原本的其他数据的索引不会发生变化，而且数组的长度也不会发生变化

逻辑中断（短路运算）

  • 表达式1 || 表达式2

    • 表达式1为真则返回真，否则返回表达式2

  • 表达式1 && 表达式2

    • 表达式1为假则返回假，否则返回表达式2

异常处理

异常的最大特征是，会报错，后面的代码无法执行

  • 语法异常 就是语法错误导致的异常啊

  • 浏览器的差异导致的异常，最常见的是IE6-8的兼容问题

  • 最常见的异常，也就是=>运行时异常。
